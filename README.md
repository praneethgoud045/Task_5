## Setup

1. Install required dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Open the Jupyter Notebook:
   ```sh
   jupyter notebook notebooks/Task5.ipynb
   ```
3. Run the notebook step by step to execute the ML workflow.
